# API Endpoints

## /stacc/v1
GET - returns number of blocks, number of blocks completed

## /stacc/v1/users/
GET - returns list of users, (id and name)

## /stacc/v1/users/id/
GET - returns list of claim IDs from that user

## /stacc/v1/claims/id/
GET - returns claim `id` (claim id, block start, block end)
